# 🧠 OCD Patients Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing OCD (Obsessive-Compulsive Disorder) patient data using Python for Exploratory Data Analysis (EDA) and Tableau for interactive dashboard visualization.

The goal is to identify patterns in patient demographics, obsession types, compulsions, and mental health conditions such as anxiety and depression.

## 🔍 Exploratory Data Analysis (EDA)
The following analysis was performed using Python:

- Obsession type distribution by gender
- Compulsion vs obsession relationship (scatter plot)
- Distribution of depression and anxiety by gender
- Age group analysis of patients
- Yearly admission trends

---

## 📈 Key Dashboard Insights

- Total Patients: **1500**
- Average Age: **58 years**
- Gender Distribution:
  - Male: ~50.2%
  - Female: ~49.8%

- Most common compulsions:
  - Washing
  - Counting

- Admissions peaked around **2018** and gradually declined afterward.
